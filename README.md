# STELLO — Personal AI OS

> A customizable AI assistant that adapts to your personality, goals, and workflow.

## Overview

STELLO is a full-stack AI-powered assistant designed to feel personal and interactive.
It combines AI responses, customizable themes, and productivity features into a single system.

## Features

*  Context-aware AI responses
*  20+ dynamic UI themes
*  Personality-driven interaction
*  Voice input support
*  Image generation
*  Streak-based engagement system
*  Chat history and export
*  Built with a focus on real-world usability

## Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Python (Flask)
* AI: OpenAI API
* Deployment: Render
